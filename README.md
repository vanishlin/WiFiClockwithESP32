# WiFiClockwithESP32

#### 这是一个使用esp32开发板，DHT11温湿度传感器，OLED屏幕 以及我自己设计的PCB板构建的wifi时钟。可以使用网页配网，同步ntp服务器，同步心悦天气实现天气预报，DHT传感器获取本地温湿度。
